# 项目介绍

## 项目简介

使用 Vue、uniapp开发 ，商城包含首页商品展示模块、商品分类模块、购物车模块，个人信息模块。。该项目主要用于学习。

## 项目运行

```
git clone https://github.com/Jasmine011/admin-back.git
在HBuilder X中打开，运行→运行到小程序模拟器→微信开发者工具，即可查看项目页面。
```
